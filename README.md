# BanikpyCo - C, ale po Ostravsku
Překladač standardních příkazů jazyka C do Ostravštiny
*(Čte se "Baník py-C-o")*

BaníkpyCo je čistě neformální rozšiřující hlavička pro jazyk C, která překládá klíčová slova a standardní funkce do ostravského dialektu.  
Není to překladač, není to nový jazyk — je to jen *krutě sprostý `#include`, co dělá C víc “z Ostravy než plyn a uhlí.”*

Příklad kódu:

```c
#include "banikpyco.h"

cyslo bo() {
    mluv("Banik pyčo!\n") pyco
    jebat 0 pyco
}
